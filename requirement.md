# Project Requirements & Environment Setup

## 1. Software Requirements
To run this Resume Screening System, you need the following environment:
* **Operating System:** Windows, macOS, or Linux.
* **Language:** Python 3.8 or higher (Tested on Python 3.13).
* **Package Manager:** pip (Python Package Installer).

## 2. Python Libraries (Dependencies)
The project relies on the following open-source libraries for Data Science and Machine Learning:

| Library | Version (Recommended) | Purpose |
| :--- | :--- | :--- |
| **Pandas** | >= 1.3.0 | Data manipulation and CSV loading. |
| **Scikit-Learn** | >= 1.0.0 | Machine Learning algorithms (Logistic Regression) and NLP tools (TF-IDF). |
| **Matplotlib** | >= 3.4.0 | Basic plotting and graph generation. |
| **Seaborn** | >= 0.11.0 | Statistical data visualization (Heatmaps). |
| **Joblib** | >= 1.1.0 | Saving and loading the trained model. |

## 3. Installation Guide
To install all required dependencies at once, run the following command in your terminal:

```bash
pip install pandas scikit-learn matplotlib seaborn joblib
